a11y toolbar demo page

This simple plugin allows you to place a floating set of controls along the left side of the screen that lets users increase the font size, turn on a high contrast mode, or desaturate (remove) all of the color for easier reading.

The toolbar installs a small cookie for each option so the choice persists through the browsing experience. The choice only applies to the site that uses the toolbar.

Colors, sizes, and positioning is easily customizable.
